# PepePet 🐸

**PepePet** — виртуальный питомец для вашего ПК. Он живёт прямо на рабочем столе, реагирует на погоду за окном, требует заботы и создаёт уютную атмосферу.

***

## Возможности

- **Реагирует на погоду**  
  Питомец подтягивает погоду в вашем городе и меняет настроение: грустит в дождь и радуется солнцу.

- **Живые реакции**  
  Питомец откликается на действия пользователя, проигрывает анимации при кормёжке, игре и сне.

- **Дайте имя**  
  Назовите питомца как захотите — имя будет отображаться в интерфейсе, а в будущем можно добавить реакцию на кличку.

- **Кормление**  
  Не забывайте кормить питомца — при низком уровне голода его настроение ухудшается.

- **Режим сна**  
  Питомец может устать. Уложив его спать, вы поможете ему восстановить энергию и поднять настроение.

- **Кастомизация (в будущем)**  
  Планируется возможность менять внешний вид, выбирать одежду и аксессуары.

***

## Зачем это нужно?

PepePet превращает скучный рабочий стол в живое, эмоциональное пространство. Это не просто игрушка — это маленький виртуальный друг, который реагирует на реальный мир, напоминает о том, что нужно отдохнуть, и создаёт ощущение уюта за компьютером, не требуя реальной заботы и ухода.

## Установка

Для запуска PepePet вам понадобится:

- Python 3.8+
- Установленные зависимости из `requirements.txt`

```bash
pip install -r requirements.txt
```

Затем запустите приложение:

```bash
python main.py
```

В настройках можно указать город (или координаты) и свой API-ключ OpenWeatherMap, чтобы питомец корректно отображал погоду.
